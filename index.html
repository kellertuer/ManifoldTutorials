The Rayleigh Quotient

For a symmetric matrix, i.e. A∈Rn×n,A=ATA\in \mathbb R^{n\times n}, A = A^\mathrm{T}A∈Rn×n,A=AT, the Rayleigh quotient is defined as

f(x)=xTAx∥x∥2. f(x) = \frac{x^\mathrm{T} A x}{\lVert x \rVert^2}. f(x)=∥x∥2xTAx​.

It can be used to compute an eigenvector to the largest eigenvalue of a matrix. Let λ1\lambda_1λ1​ denote the largest eigenvalue. Then we have

λ1=max⁡x∈Rnf(x). \lambda_1 = \max_{x\in\mathbb R^n} f(x). λ1​=x∈Rnmax​f(x).

Since optimization problems are usually phrased as minimization problems, we look at

min⁡x∈Rn−f(x). \min_{x\in\mathbb R^n} -f(x). x∈Rnmin​−f(x).

Still, Optimization on the whole Euclidean space has the problem, that the minimizer x∗x^*x∗, where this minimum is attained, is not unique. This can easily be seen by observing that any a⋅xa\cdot xa⋅x, with a positive a>0a>0a>0 has the same function value, and also any such vector is of course also an eigenvector to λ1\lambda_1λ1​.

A remedy is, to restrict the function fff to all unit vectors, or in other words the unit sphere

Sn−1:={x∈Rn∣∥x∥=1} \mathbb S^{n-1} := \bigl\{x\in\mathbb R^n \big| \lVert x \rVert = 1 \bigr\} Sn−1:={x∈Rn∣∣∣​∥x∥=1}

of dimension n=1n=1n=1. The code on the left now formulates the cost function and its gradient on this Riemannian manifold in order to perform a Riemannian Quasi Newton method.

Note In Manopt.jl the example is written using the Riemannian gradient. In Manopt the Euclidean gradient, that is the on in the embedding of the sphere, is provided and internally converted via projection to the Riemannian one.

using Manopt, Manifolds, Random, LinearAlgebra
# Generate random problem data.
n = 1000
A = randn(n, n)
A = (A + A') / 2

# Create the problem structure.
M = Sphere(n - 1)

# Define the problem cost function and its Riemannian gradient.
F(X::Array{Float64,1}) = -X' * A * X
∇F(X::Array{Float64,1}) = -2 * (A * X - X * X' * A * X)

# Solve
x = random_point(M)
quasi_Newton(M, F, ∇F, x; debug=[:Iteration, " ", :Cost, "\n", 1, :Stop])



% Generate random problem data.
n = 1000;
A = randn(n);
A = .5 * (A + A.');

% Create the problem structure.
manifold = spherefactory(n);
problem.M = manifold;

% Define the problem cost function and its Euclidean gradient.
problem.cost = @(x) -x' * (A * x);
problem.egrad = @(x) -2 * A * x; % notice the 'e' in 'egrad' for Euclidean

% Solve.
[x, xcost, info, options] = trustregions(problem);
